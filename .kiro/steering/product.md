# Product Overview

## PTC Agency Demo

A Spring Boot-based travel agency management system demonstrating modern Java enterprise architecture patterns. The application manages customer data and follows Domain-Driven Design (DDD) principles with CQRS architecture.

**Key Features:**
- Customer management with comprehensive validation
- Multi-profile configuration (dev, prod)
- PostgreSQL database integration
- RESTful API with OpenAPI documentation
- Event-driven architecture with domain events

**Business Domain:**
Travel agency operations focusing on customer lifecycle management, including registration, profile management, and document validation for travel services.
